pallio demo


animated lazy load images using IntersectionObserver:


just another lazy load image respository; this one is has an animated placeholder and is for usage of IntersectionObserver API wich is more and more supported by browsers.


check out github for documentation


the idea and the core of the code is based on Craig Buckler's progressive-image.js.


in case you don't want to polyfill IntersectionObserver and requestAnimationFrame, consider using progressive-image.js instead.


this is happening:


  • A tiny version of the final image serves as a placeholder.
  • As soon as the image is in browser's view, the original image is loaded and replaced after finishing loading.
  • While the image is loading, a animation is shown on the placeholder image until the loading is finished.
  • The animation is a rounded box that moves to the edges, changes its color and.
  • As the new image is revealed, a box shadow is added to the newly added image.

Scroll down the page in order to experience the visual effect.
